## Problema

Um sistema financeiro precisa notificar múltiplos investidores quando o preço de
ações muda. O código atual faz polling constante ou tem dependências diretas
entre as ações e os investidores, criando acoplamento forte e código difícil de
manter.

## Sobre o CarnaCode 2026

O desafio **CarnaCode 2026** consiste em implementar todos os 23 padrões de
projeto (Design Patterns) em cenários reais. Durante os 23 desafios desta
jornada, os participantes são submetidos ao aprendizado e prática na
idetinficação de códigos não escaláveis e na solução de problemas utilizando
padrões de mercado.
